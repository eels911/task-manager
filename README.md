# Task manager
В данной лабораторной работе нашей командой реализован сайт для создания заметок.

## Функционал:
авторизация;  
регистрация;  
создание, просмотр, редактирование, удаление карточки;  
изменение статуса карточки.  

## Разработка
$ git clone  
$ npm install  
$ npm start
